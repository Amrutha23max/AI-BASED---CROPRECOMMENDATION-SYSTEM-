## Description

**1.Start Analysis:** This is the starting point where the user begins the crop recommendation process.

**2.Choose Input Method:** The user selects how to provide soil data, either by uploading an image or entering details manually.

**3.Upload Soil Image:** The user uploads a soil image for automatic analysis by the system.

**4.Enter Soil Details:** Alternatively, the user can manually input soil parameters such as soil type and nutrients.

**5.Submit:** After providing the input, the user submits the data for processing.

**6.View Recommended Crops:** The system analyzes the input and displays the most suitable crops based on soil and environmental conditions.

**7.View Explanation:** The system provides an explanation of the recommendations to help the user understand the decision-making process.
